# Atelier Web

Repo **interne** de l'activité de création de sites web (méthode + maquettes de prospection).
Distinct du repo client `mj-atlantique-mockup` (qui ne contient que le site de Delphine Raymond).

## Organisation
- **`methode/`** — la méthode réutilisable :
  - `PLAYBOOK.md` — process de vente/livraison (professions juridiques + au-delà)
  - `GRILLE-AUDIT-WEB.md` — grille d'audit d'un site prospect (score /44 + quick wins)
  - `EXPERIENCE-PROSPECTION.md` — protocole d'expérience « quel segment attaquer » + données passe 1
- **`mockups/`** — les maquettes de prospection (le « cadeau » personnalisé par prospect) :
  - `sylvie-janus.html` — pilote, Institut Sylvie Janus (Champel, Genève)

## Publication
GitHub Pages sert ce repo → chaque maquette a une URL « cadeau » à montrer/envoyer au prospect :
`https://benjiidelpech.github.io/atelier-web/mockups/<prospect>.html`

> Les maquettes sont des **previews de démonstration** (non officielles), à la marque du prospect, pour illustrer une refonte. Voir le commentaire en bas de chaque fichier.

## Pipeline
Le suivi des prospects (audit → mockup → contact → RDV → signé) est tenu dans le Google Sheet
« Expérience prospection sites web » (lien dans `methode/EXPERIENCE-PROSPECTION.md` / Drive).
